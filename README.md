# Weave

A clean, visual workspace for organizing files, documents, and ideas.

Weave combines a familiar file explorer with a freeform canvas. Arrange files and
folders visually, group and connect items, or open a document view to read and edit
content. The workspace runs locally in the browser and supports uploads, previews,
and downloads.

## Features

- **Local-first** — your workspace and changes stay in your browser, so no account
  or backend is required.
- **Interactive canvas** — drag, resize, group, and connect files, folders, notes,
  and workers in a visual workspace.
- **Free-form file viewing** — switch between canvas, explorer, and document views
  instead of being restricted to a fixed folder layout.
- **Document support** — preview PDFs and work with Markdown, code, JSON, and text
  files, with upload and download support.
